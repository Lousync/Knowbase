每天可以写一篇博客文章，使用 **Markdown** 格式编写。

## 创建文章

- 点击侧栏 **今日文章编写** 创建今天的文章
- 在侧栏「文章归档」树中点击任意日期创建 / 打开对应日期的文章
- 使用侧栏搜索框输入日期快速定位（支持 `2026/06/17` 等格式）

## 编辑与保存

- 编辑器支持 Markdown 实时预览（点击工具栏 👁 按钮或按 `Ctrl` `/`）
- 每 2 秒自动保存一次，也可随时按 `Ctrl` `S` 手动保存
- 顶部工具栏的圆点指示灯：**黄色 = 未保存，绿色 = 已保存**

## 日期限制

- **不能创建未来日期的文章**
- 支持历史日期补写（有最早日期限制）
- 已有内容的未来日期文章仍可正常编辑和删除

## Markdown 基础

| 语法 | 效果 |
|------|------|
| `**粗体**` | **粗体** |
| `*斜体*` | *斜体* |
| `# 标题` | 一级标题 |
| `- 列表` | 无序列表 |
| `[文字](url)` | 链接 |
| `![alt](url)` | 图片 |
