## 全局

| 快捷键 | 功能 |
|--------|------|
| `Ctrl` `B` | 折叠 / 展开侧栏 |
| `Escape` | 关闭弹窗 / 返回列表 |

## 知识库

| 快捷键 | 功能 |
|--------|------|
| `Ctrl` `N` | 新建零散页面 |
| `Ctrl` `W` | 关闭当前 Tab |
| `Ctrl` `Tab` | 下一个 Tab |
| `Ctrl` `Shift` `Tab` | 上一个 Tab |
| `F2` | 重命名选中项 |
| `Delete` | 删除选中项（页面 > 章节 > 目录） |
| `Ctrl` `S` | 立即保存 |
| `Ctrl` `/` | 切换 Markdown 预览 |

## 博客

| 快捷键 | 功能 |
|--------|------|
| `Ctrl` `N` | 新建 / 打开今日文章 |
| `Ctrl` `S` | 保存并关闭 |
| `Ctrl` `/` | 切换预览 |
| `Escape` | 返回列表 |
| `Delete` | 删除文章 |

## 日程

| 快捷键 | 功能 |
|--------|------|
| `Ctrl` `N` | 新建任务 |
| `Escape` | 关闭弹窗 |
