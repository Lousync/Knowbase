使用 Emoji 表情记录每天的心情状态，支持多选，也可不选。

## Emoji 一览

| Emoji | 含义 | 说明 |
|-------|------|------|
| 😄 | 开心 | 积极愉快的情绪 |
| 😫 | 疲倦 | 精力耗尽、需要休息 |
| 😢 | 难过 | 悲伤或失落的情绪 |
| 😕 | 困惑 | 对某事感到迷茫或不确定 |

## 使用方法

在博客编辑器顶部的工具栏中，点击你想记录的心情 Emoji 即可切换选中/取消。选中的 Emoji 会以正常透明度显示，未选中的为半透明。

状态会实时自动保存，并展示在博客卡片上。
